Snippet Head

Einbindung des Snippets:

<?php snippet('head') ?>

Das Snippet Head dient zur Einbindung des HTML <head></head> mit einer Reihe an Konfigurationsmöglichkeiten. Aktuell können bei der Einbindung Meta-Daten gesetzt werden.

erweiterte Konfiguration

<?php 
	$options = [
		'robots' => '',
		'metaDescription' => '',
		'title' => ''
	]
?>

<?php snippet('head', $options) ?>

CSS Ladeverhalten

CSS wird durch das Head Snippet in folgender Reihenfolge geladen:

  1. index.css ( aus dem Ordner assets/css/index.css )
  1. <template>.css (templatespezifisches CSS: assets/css/templates/<template>.css)

Snippet Header

<?php snippet('header') ?>

Generelle Navigation

Ohne zusätzliche Konfiguration wird der Standard Header mit den spezifizierten Menüseiten geladen.

One-Page Navigation

Für die Konfiguration einer On-Page Navigation sind diverse Optionen notwendig. Zum Einen ist die Navigations-Link-Struktur zu definieren, zum Anderen die Einstellung für Single Page zu treffen.

<?php 

	// Links für die Navigation
	$linkConfig = [
	  ['href' => '#programm', 'title' => 'Programm'],
		...
	];

	$options = [
		'singlePage' => true,
		'linkConfig' => $linkConfig
	];
?>

<?php snippet('head', $options) ?>

Das <main> Element muss wie folgt im Template definiert werden.

<main data-syp="scroll" data-target="#scrollTarget" data-offset="88">
	<article id="programm">...</article>
</main>

Sofern die Abschnitte im Template mit den entsprechenden IDs versehen sind initialisiert sich das Scrolling automatisch und die Navigations-Links werden im jeweiligen Seitenabschnitt hervorgehoben.

Snippet Footer

<?php snippet('footer') ?>

JavaScript Ladeverhalten

JS wird durch das Footer Snippet in folgender Reihenfolge geladen:

  1. index.js ( aus dem Ordner assets/js/index.css )
  1. <template>.js ( templatespezifisches CSS: assets/js/templates/<template>.js)

Zusätzliche Skripte

<?php 
	$additionalScripts = [
		$script = '<script ...> ...content </script>',
		$scriptFromFils = js('assets/js/custom/my-custom-script.js')
	]
?>

<?php snippet('head', ['additionalScripts' => $additionalScripts]) ?>

CSS Autoloading

Laden durch Template Name

Durch das Snippet wird CSS/JS für Templates automatisch geladen wenn

  1. die CSS/JS Datei im Verzeichnis assets/css oder js/templates liegt
  1. und die CSS/JS Datei den gleichen Namen wie das Template trägt

Als Beispiel: sample-template.php und sample-template.css.
